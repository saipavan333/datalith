📥 Read & write

import pandas as pd

df = pd.read_csv("f.csv", dtype={"id":"int64"},
                 parse_dates=["ts"])
df = pd.read_parquet("f.parquet")      # columnar, fast
df = pd.read_sql(query, conn)

Prefer Parquet over CSV: typed, columnar, compressed, far faster.

🔎 Filter (= WHERE)

df[(df.amt > 100) & (df.dept == "eng")]   # & | ~, wrap ()

df[df.name.str.startswith("A")]

Transform

➕ New columns

df["net"] = df.amt - df.fee          # vectorized
df = df.assign(net=lambda d: d.amt - d.fee)

import numpy as np
df["tier"] = np.where(df.amt > 100, "big", "small")
df["g"] = pd.cut(df.amt, [0,10,100,np.inf],
               labels=["s","m","l"])
df["m"] = df.dept.map({"eng":1})       # lookup

Vectorize with column ops / np.where; avoid iterrows / apply(axis=1) on big frames.

🧹 Missing data & dtypes

df.isna().sum()             # nulls per col
df.dropna(subset=["amt"])
df.fillna({"amt": 0})
df["amt"] = df.amt.fillna(df.amt.median())

df["id"] = df.id.astype("int64")
pd.to_numeric(df.x, errors="coerce")   # bad -> NaN
df.drop_duplicates(subset=["id"], keep="last")

Σ GroupBy (= GROUP BY)

df.groupby("dept").amt.sum()
df.groupby("dept").agg(
    n=("id", "count"),
    total=("amt", "sum"),
    avg=("amt", "mean")).reset_index()

df.groupby(["dept","month"]).amt.sum()
df.groupby("dept").amt.transform("mean")  # broadcast back

🔗 Merge / join / concat

pd.merge(a, b, on="id", how="left")   # inner|left|right|outer
pd.merge(a, b, left_on="aid", right_on="id")
a.join(b, on="id")                    # on index

pd.concat([a, b])              # stack rows (UNION ALL)
pd.concat([a, b], axis=1)      # side-by-side
pd.merge(a, b, on="id", indicator=True) # _merge col

🪟 Sort, rank & window

🧭 SQL → pandas map

  • WHERE → boolean mask df[df.x>0]
  • GROUP BY → df.groupby().agg()
  • JOIN → pd.merge()
  • ORDER BY → sort_values()
  • UNION ALL → pd.concat()
  • DISTINCT → drop_duplicates()
  • window → groupby().transform()/rank()/rolling()

⚡ Performance & scale

  • Set dtypes (category for low-cardinality strings) to cut memory.
  • Read only needed columns: read_parquet(cols=[...]).
  • Process in chunks: pd.read_csv(..., chunksize=100_000).
  • Too big for RAM? Switch to Polars, DuckDB, or Dask.
  • Method-chain with .pipe() for readable transforms.
