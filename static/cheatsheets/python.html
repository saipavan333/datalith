⏳ Iterators & generators

def read_big(path):
    with open(path) as f:
        for line in f:
            yield line.strip()   # lazy, O(1) memory

for row in read_big("huge.csv"):
    process(row)

Generators stream data larger than RAM — never f.readlines() a huge file.

📅 datetime

from datetime import datetime, timezone, timedelta

now = datetime.now(timezone.utc)   # UTC-aware
datetime.strptime(s, "%Y-%m-%d")   # parse
now.strftime("%Y-%m-%d %H:%M")      # format
now + timedelta(days=7)
now.isoformat()   now.date()

Always store/compute in UTC; localize only for display.

🌐 HTTP & DB ingestion

import requests
r = requests.get(url, params={...}, timeout=30)
r.raise_for_status(); data = r.json()
# paginate + backoff on 429/5xx

from sqlalchemy import create_engine, text
eng = create_engine("postgresql://…")
with eng.connect() as c:
    c.execute(text("SELECT :x"), {"x":1})  # parametrized!

Never f-string values into SQL — parametrize (SQL-injection & typing).
