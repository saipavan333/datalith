matplotlib

🖼️ Figure & axes

import matplotlib.pyplot as plt

fig, ax = plt.subplots(figsize=(8, 4))
ax.plot(x, y)
ax.set_title("Sales")
ax.set_xlabel("date"); ax.set_ylabel("amt")
plt.tight_layout()
plt.show()

Prefer the object-oriented API (fig, ax) over stateful plt.plot() — clearer for multi-panel figures.

🔲 Subplots

fig, axes = plt.subplots(2, 2, figsize=(10, 8))
axes[0, 0].plot(x, y)
axes[0, 1].hist(data)
for ax in axes.flat:
    ax.grid(True, alpha=.3)
fig.suptitle("Dashboard")
fig, ax = plt.subplots(1, 3, sharey=True)

🎨 Style & annotate

ax.plot(x, y, color="steelblue", lw=2,
        linestyle="--", marker="o")
ax.legend(loc="upper left")
ax.set_xlim(0, 100)   ax.set_ylim(0, 1)
ax.set_xticks(range(0, 101, 20))
ax.tick_params(rotation=45)
ax.axhline(0, color="gray")   ax.annotate("peak", (x0, y0))

🐼 pandas shortcut

df.plot(x="date", y="amt")          # uses matplotlib
df.plot.bar()   df.plot.hist(bins=30)
df.plot.scatter(x="a", y="b")
df.groupby("dept").amt.sum().plot.bar()
ax = df.plot(); ax.set_title("…")   # returns an Axes
Seaborn

🌊 Setup & themes

import seaborn as sns
sns.set_theme(style="whitegrid", palette="deep")

# Seaborn takes a DataFrame + column names
sns.scatterplot(data=df, x="amt", y="fee",
                hue="dept", size="qty")
sns.lineplot(data=df, x="date", y="amt", hue="region")

hue/size/style map columns to color/size/marker — grouped plots in one line.

📊 Categorical & distribution

sns.barplot(data=df, x="dept", y="amt")   # mean + CI
sns.boxplot(data=df, x="dept", y="amt")
sns.violinplot(data=df, x="dept", y="amt")
sns.countplot(data=df, x="status")

sns.histplot(data=df, x="amt", bins=30, kde=True)
sns.kdeplot(data=df, x="amt", hue="dept")
sns.ecdfplot(data=df, x="amt")

🔥 Relationships & grids

# correlation heatmap
sns.heatmap(df.corr(numeric_only=True),
            annot=True, cmap="coolwarm")
sns.pairplot(df, hue="dept")      # scatter matrix

# faceting: one panel per category
sns.relplot(data=df, x="date", y="amt",
            col="region", row="channel", kind="line")
sns.catplot(data=df, x="dept", y="amt", kind="box", col="year")

💡 Tips

  • Seaborn returns matplotlib objects — refine with ax.set_* after plotting.
  • figure-level (relplot/catplot/displot) do faceting; axes-level (scatterplot/boxplot) draw on one ax.
  • Aggregate first for big data (plotting millions of points is slow/unreadable).
  • Save vector (.svg/.pdf) for reports; dpi=150+ for crisp PNGs.
