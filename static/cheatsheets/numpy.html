Create & inspect

🧱 Create arrays

import numpy as np

np.array([1, 2, 3])            # from list
np.array([[1,2],[3,4]])        # 2-D
np.zeros((2,3))   np.ones(4)   np.full((2,2), 7)
np.arange(0, 10, 2)            # 0 2 4 6 8
np.linspace(0, 1, 5)          # 5 evenly spaced
np.eye(3)                      # identity
np.random.default_rng(0).random((2,2))

🔍 Attributes & dtype

a.shape      # (rows, cols)
a.ndim       # number of dims
a.size       # total elements
a.dtype      # int64, float64, ...
a.astype(np.float32)          # cast
a.nbytes     # memory used
np.array([1,2], dtype=np.int32)

A NumPy array is homogeneous (one dtype) and contiguous → C-speed vectorized ops.

🎯 Index & slice

a[0]        a[-1]        a[1:4]
a[2, 3]                  # 2-D element (row, col)
a[1, :]     a[:, 0]      # row / column
a[a > 5]                 # boolean mask
a[[0, 2, 4]]             # fancy indexing
a[a % 2 == 0] = 0        # masked assignment

Slices are views (share memory); fancy/boolean indexing returns copies.

🔄 Reshape & combine

a.reshape(3, 4)     a.reshape(-1, 2)   # -1 = infer
a.ravel()   a.flatten()      # to 1-D
a.T                          # transpose
np.concatenate([a, b], axis=0)
np.vstack([a, b])   np.hstack([a, b])
np.stack([a, b])    np.split(a, 3)
Compute

➗ Vectorized math

a + b   a * 2   a ** 2   a / b     # elementwise
np.sqrt(a)  np.exp(a)  np.log(a)
np.sin(a)   np.abs(a)  np.round(a, 2)
np.maximum(a, b)   np.clip(a, 0, 100)
a @ b        # matrix multiply (dot)

Always vectorize — no Python for loops over elements. That's the whole point of NumPy.

📡 Broadcasting

a = np.ones((3, 4))
a + np.array([1, 2, 3, 4])   # row added to each row
a * np.array([[1],[2],[3]])  # column scales each row
# rule: dims align from the right; size 1 stretches
(3,4) + (4,)   -> ok
(3,4) + (3,1)  -> ok
(3,4) + (3,)   -> error

Σ Aggregations (axis)

a.sum()   a.mean()   a.std()   a.min()  a.max()
a.sum(axis=0)   # down columns -> per-column
a.sum(axis=1)   # across rows -> per-row
a.cumsum()      a.argmax()   a.argmin()
np.median(a)    np.percentile(a, 95)
np.nanmean(a)   # ignore NaN

axis=0 collapses rows (result per column); axis=1 collapses columns (result per row).

🧭 Logic & where

np.where(a > 0, "pos", "neg")   # vectorized if/else
np.select([a<0, a>0], ["n", "p"], default="z")
np.any(a > 5)    np.all(a > 0)
np.unique(a, return_counts=True)
np.isnan(a)     np.isin(a, [1,2,3])

🧮 Linear algebra & random

np.dot(a, b)     a @ b
np.linalg.inv(m)     np.linalg.det(m)
np.linalg.solve(A, b)                 # Ax = b
np.linalg.eig(m)     np.linalg.norm(v)

rng = np.random.default_rng(42)
rng.random(5)   rng.integers(0, 10, 5)
rng.normal(0, 1, (3,3))   rng.choice(a, 3)

🔀 Sort & set

np.sort(a)          np.argsort(a)   # indices
a[np.argsort(a)]                    # sort by key
np.unique(a)        np.intersect1d(a, b)
np.union1d(a, b)    np.setdiff1d(a, b)
np.searchsorted(sorted_a, v)

⚡ Performance

  • Vectorize; avoid Python loops & np.append in a loop (reallocates).
  • Preallocate with np.empty/zeros, then fill.
  • Use views (slices) to avoid copies; be aware they alias memory.
  • Pick the smallest dtype that fits (float32, int32) to cut memory/bandwidth.
  • Interops zero-copy with pandas/Arrow/PyTorch via the buffer protocol.
